# Agriculture Farm Management System

This project is a simple web-based Agriculture Farm Management System developed for Web Lab purposes. It helps in managing basic farm-related information such as crops, farm activities, and agricultural data.

## Features
- Crop Information Management
- Farm Activity Tracking
- Simple User Interface
- Responsive Web Design
- Basic Data Handling using JavaScript

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Purpose
The main objective of this project is to design and develop a simple web-based system for managing agricultural farm data and to practice web development skills.

## Project Structure
```text
Project Folder/
│── index.html
│── style.css
│── script.js
│── README.md
```

## How to Run
1. Download or clone the project.
2. Open `index.html` in any web browser.
3. Use the system to view and manage farm data.
